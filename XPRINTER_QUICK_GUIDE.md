### **Printer not showing in list?**
- Check USB cable connected
- Check printer is powered ON
- Restart browser

### **Receipt is blank?**
- Check thermal paper installed correctly (thermal side up)
- Clean print head with alcohol wipe

### **Receipt is cut off?**
- Set paper size to **80mm x Auto** (not 80mm x 80mm)
- Set orientation to **Portrait**

### **Print dialog blocked?**
- Allow popups in browser
